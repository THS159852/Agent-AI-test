# Risk Analyzer

## Risk categories

- Requirement ambiguity
- Integration / third-party dependency
- Security & privacy
- Data integrity & migration
- Performance & scale
- Environment / test data availability
- Schedule & resource constraints

## Output template

```markdown
## Risk Register
| ID | Risk | Category | Likelihood | Impact | Mitigation | Test focus |
|----|------|----------|------------|--------|------------|-----------|
```
